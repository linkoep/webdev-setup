Using Babel as a Transpiler

A transpiler like Babel can help us ensure that our new JavaScript features remain compatible with older browsers.
By installing babel via npm, and adding it's loader to our webpack build step with some extra rules,
we can translate (transpile) our new features into "regular" JavaScript.
We can also see this by searching through our bundle.js file to find that our use of a template string has been converted to normals tring concatenation
