Configuring Webpack

When we started using webpack, we had to manually specify the bundling entry point:
Our main javascript file that used "require" to include other libraries,
As well as the filename we wanted for our output.
We can automate this process by creating the "webpack.config.js" file, which allows us to simply run webpack without having to worry about various arguments.
This file also allows us to add other advance features to our build step.
